## 🚀 Featured Projects

### 🧩 [Multiplayer Minesweeper](https://github.com/OntarioTech-CS-program/w25-csci2020u-finalproject-w25-team30) &nbsp;`Java` `TCP Sockets` `Swing`
> **Real-time multiplayer game engine built from scratch over raw TCP sockets**

- Designed and implemented a **custom application-layer protocol** (CREATE / JOIN / COORD / MINE / GAMEOVER) to synchronize game state across clients without a third-party framework.
- Built **server-side lobby management** with deterministic board seeding and concurrent per-client handler threads using `synchronized` shared state — no race conditions.
- Developed a full **Java Swing desktop UI** with lobby flows, game ID clipboard sharing, and live board rendering.

---

### 🎬 [MovieFlix](https://github.com/Dior15/BlueCircuit) &nbsp;`Python` `Flask` `SQLAlchemy` `SQLite` `TMDb API`
> **Full-stack movie discovery platform with auth, watchlists, and external API integration**

- Integrated the **TMDb REST API** to serve trending, top-rated, and search results with pagination and multi-category filters — handling real-world rate-limited, paginated responses.
- Implemented **user authentication** (password hashing via Werkzeug) and a persistent watchlist using SQLAlchemy ORM with a normalized SQLite schema.
- Wrote a **pytest suite** covering auth flows, API error handling, and database mutations — reinforcing correctness at the integration level.

---

### 🏋️ [FitTrack](https://github.com/Dior15/FitTrack) &nbsp;`Flutter` `Dart` `sqflite` `Open Food Facts API`
> **Cross-platform fitness tracker with offline-first architecture and custom data visualizations**

- Architected an **offline-first SQLite schema** (users / foods / exercises / records) supporting the full app lifecycle without a network connection.
- Built **custom dashboard charts** using Flutter's `CustomPainter` API — zero external charting libraries.
- Integrated the **Open Food Facts API** for nutritional search, scheduled meal reminders via local notifications, and a unified activity feed across food and exercise logs.
